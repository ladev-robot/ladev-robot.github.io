ROS Overview and Environment Setup – A Beginner’s Guide to Getting Started

Posted on 2025-08-14 Edited on 2025-08-18 In 机器人 , 虚拟机， ROS

ROS概述

  ROS 具有多个版本，可以在 ubuntu、Mac 或 windows10 系统上安装，虽然一般用户平时使用的操作系统以windows居多,但是ROS的版本基本都不支持windows。所以绝大多数应用场景都是在Ubuntu，但想要在windows上运行ROS也是可以办到的。下面介绍三个在Ubuntu安装常用方式:

  • 直接安装或双系统安装Ubuntu
  • 虚拟机安装Ubuntu
  • Windows使用WSL安装Ubuntu
    三种方式比较，各有优缺点：
  • 方法一可以保证性能，且不需要考虑硬件兼容性问题，但是和windows系统交互不便；
  • 方法二可以方便的实现 windows 与 ubuntu 交互，不过性能稍差，且与硬件交互不便。
  • 方法三可以方便的实现 windows 与 ubuntu 交互，且支持硬件交互，但会出现意想不到的Bug。

 在 ROS 中，一些仿真操作是比较耗费系统资源的，且经常需要和一些硬件(雷达、摄像头、imu、STM32、arduino….)交互，因此，原则上建议采用方案1，不过如果只是出于学习目的，那么方案2也基本够用，且方案2在windows与ubuntu的交互上更为方便，对于学习者更为友好，因此本教程在此选用的是方案2。如果是初学对Linux环境部署不熟悉也可以选择方案3。当然，具体采用哪种实现方案，请按需选择。

如果采用虚拟机安装 ubuntu，再安装 ROS 的话，大致流程如下:

  1. 安装虚拟机软件(比如：virtualbox 或 VMware)；
  2. 使用虚拟机软件虚拟一台主机；
  3. 在虚拟主机上安装 ubuntu ；
  4. 在 ubuntu 上安装 ROS；
  5. 测试 ROS 环境是否可以正常运行。

 以下是Ubuntu和ROS对应的版本和安装命令，不同的Ubuntu版本对应这不同的ROS版本，一定要对应版本！！！

1.1 VMWare安装

VMware在2024年推出了两款免费的版本版本 Workstation Pro 17.5.2 和 Fusion Pro 13.5.2，本文以安装Workstation Pro 17.5.2为例。

PS：由于VMware官网是国外网站，全程最好开启梯子。

一. 进入官网

VMWARE官网

三. 注册成功后回到下载界面再次点击下载

  2. 在搜索框输入VMware Workstation Pro

  3. 点击Search

四. 点击Downloading VMware Workstation Pro，下拉页面选择Download VMware Workstation Pro (Free)

五. 下载对应的版本

1.2 创建虚拟机

一. 下载Ubuntu镜像，这里给出中科大的镜像源，大家自行选择下载，根据本人经验，大部分开源项目支持Ubuntu20.04版本，之后的内容也是以Ubuntu20.04为例。Ubuntu镜像源

二. 新建虚拟机，自定义虚拟机配置

三. Vmware会自动识别电脑中下载的镜像文件，一直点下一步。

四. 设置好存储位置和用户信息后，对虚拟机的CPU和内存进行分配，分配给虚拟机CPU的处理器内核总数 < CPU的逻辑处理器(线程数)，建议为CPU的$\frac{1}{2} \text{ - } \frac{2}{3}$之间。内存建议$\frac{1}{2}$左右。

五. 网络模式和磁盘空间根据个人需求和电脑配置自行决定。推荐使用桥接模式。

  • 桥接模式：能够连接外网，使用单独的ip地址，网段需要与桥接连接的外部网络的网段相同。
  • NAT模式：使用物理机的网络。

六. 点击完成即可，相关配置后续也可修改。

七. 上述步骤完成后，等待一段时间。即可进入Ubuntu系统
