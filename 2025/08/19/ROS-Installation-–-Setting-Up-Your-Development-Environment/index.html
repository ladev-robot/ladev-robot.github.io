ROS安装

Ubuntu 安装后，安装 ROS 操作系统，大致步骤如下:

  1. 配置ubuntu的软件和更新；
  2. 设置安装源；
  3. 设置key；
  4. 安装；
  5. 配置环境变量。

对于熟悉ROS的同学可以直接使用鱼香ROS的一键安装命令，省去下面的步骤。对于刚接触的同学来说，建议按照下面步骤逐步安装。

3. 设置key

使用密钥来验证软件包源的身份和完整性，以确保从可信任的源安装软件。

4. 安装

首先需要更新 apt(以前是 apt-get, 官方建议使用 apt 而非 apt-get),apt 是用于从互联网仓库搜索、安装、升级、卸载软件或操作系统的工具。

然后，再安装所需类型的 ROS:ROS 多个类型:Desktop-Full、Desktop、ROS-Base。这里介绍较为常用的Desktop-Full(官方推荐)安装: ROS, rqt, rviz, robot-generic libraries, 2D/3D simulators, navigation and 2D/3D perception

以下是Ubuntu对应的ROS版本

Ubuntu ROS1 命令
14.04 LTS Indigo lgloo sudo apt install ros-indigo-desktop-full
16.04 LTS Kinetic Kame sudo apt install ros-kinetic-desktop-full
18.04 LTS Melodic Morenia sudo apt install ros-melodic-desktop-full
20.04 LTS Noetic Ninjemys sudo apt install ros-noetic-desktop-full

1.1.2 ROS测试

  1. 首先启动三个命令行(ctrl + alt + T)
  2. 命令行1键入:roscore
  3. 命令行2键入:rosrun turtlesim turtlesim_node(此时会弹出图形化界面)
  4. 命令行3键入:rosrun turtlesim turtle_teleop_key(在3中可以通过上下左右控制2中乌龟的运动)

注意：光标必须聚焦在键盘控制窗口，否则无法控制乌龟运动。
